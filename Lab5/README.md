# Lab5 - Работа с фреймворком React

## 🚀 Как запустить

### Локально (через npm)

1. Установите зависимости:
```bash
npm install
```

2. Запустите проект:
```bash
npm run dev
```

3. Откройте браузер: http://localhost:5173

### На CodeSandbox (онлайн)

**Способ 1: Импорт из GitHub**
1. Перейди на https://codesandbox.io/
2. Нажми "Import from GitHub"
3. Вставь URL репозитория

**Способ 2: Загрузка ZIP**
1. Создай ZIP архив папки Lab5
2. Перейди на https://codesandbox.io/
3. Нажми "Import" → "From ZIP"
4. Загрузи архив

**Способ 3: Вручную**
1. Перейди на https://codesandbox.io/
2. Создай новый Sandbox: "Vite + React"
3. Загрузи все файлы из Lab5

📖 **Подробная инструкция**: см. файл `CODESANDBOX_GUIDE.md`

## 📚 Выполненные упражнения

### 01 - Начало
Базовый рендеринг с использованием переменных в JSX

### 02 - Рендер объектов
Работа с объектами и их свойствами в JSX

### 03 - Построение макета
Создание Bootstrap карточки с данными из объекта

### 04 - Построение из массивов
Рендеринг массива HTML элементов

### 05 - Отображение массива в li
Использование map() для преобразования массива в список

### 06 - Отображение массива объектов в li
Рендеринг списка планет с использованием Bootstrap list-group

### 07 - Рендеринг с помощью функций
Создание функций, возвращающих JSX

### 08 - Первый функциональный компонент
Создание компонентов BootstrapCard и Hero с PropTypes

### 09 - Условный рендеринг
Компонент Alert с условным отображением и разными цветами

### 10 - Добавление стилей
Использование inline стилей в React

### 11 - Прослушивание событий
Обработка событий onClick в компонентах

## 🛠 Технологии

- React 18
- Vite
- Bootstrap 5
- PropTypes
