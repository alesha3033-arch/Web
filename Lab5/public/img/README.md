# Папка для изображений

## Текущие изображения:

- `Dylan.png` - изображение Bob Dylan (используется в упражнениях 3 и 8)

## Как использовать в коде:

В любом компоненте используйте путь `/img/имя_файла`:

```jsx
<img src="/img/Dylan.png" alt="Bob Dylan" />
```

## Примечание:

Папка `public` в Vite доступна напрямую через `/`, поэтому:
- `public/img/Dylan.png` → `/img/Dylan.png` в коде

## Как добавить свое изображение:

1. Скопируйте ваше изображение в эту папку `Lab5/public/img/`
2. Используйте в коде: `<img src="/img/ваше_фото.jpg" />`
