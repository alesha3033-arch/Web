# Web Development Labs

Лабораторные работы по веб-разработке (4-6)

## Структура проекта

- **Lab4** - JavaScript игра
- **Lab5** - React приложение
- **Lab6_Movies** - React приложение с фильмами
- **Lab6_Variant7** - Вариант 7 лабораторной работы 6

## Запуск проектов

### Lab5 (React)
```bash
cd Lab5
npm install
npm start
```

### Lab6_Movies (React)
```bash
cd Lab6_Movies
npm install
npm start
```

### Lab6_Variant7 (React)
```bash
cd Lab6_Variant7
npm install
npm start
```
